## Common Issues & Fixes

### 1. Computer Not Connecting to Wi-Fi
**Possible Causes:**
- Airplane mode enabled
- Router issues
- Network driver problem

**Fixes:**
- Restart the computer
- Restart the router
- Forget and reconnect to the Wi-Fi network

### 2. Computer Running Slow
**Possible Causes:**
- Too many startup programs
- Low storage
- Malware

**Fixes:**
- Disable startup apps
- Free up disk space
- Run antivirus scan


### 3. Application Not Responding
**Fixes:**
- Close the app using Task Manager
- Restart the computer
- Reinstall the application
